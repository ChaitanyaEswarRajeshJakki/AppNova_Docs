## How the conversions were produced
- **DOCX:** `pandoc <md> -f gfm --toc --toc-depth=2 --metadata title=... -o <docx>`.
- **PDF:** `pandoc <md> -f gfm -s --toc --embed-resources -c handbook_style.css -o handbook.html`,
  then a Playwright (Python 3.11 / Chromium) script prints the HTML to A4 PDF with print
  background, CSS `@page` sizing, and a page-number footer. No LaTeX / weasyprint / wkhtmltopdf
  engine was available, so Chromium was used as the PDF engine.
- Helper files live in the session scratchpad (`handbook_style.css`, `html_to_pdf.py`), not
  committed.
